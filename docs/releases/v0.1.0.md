# Release Notes — v0.1.0

## Overview

Initial release of **Dutch Language**, a static browser-based vocabulary practice app for Dutch inburgering-style reading vocabulary, focused on formal letters, signaalwoorden, procedures, conditions, obligations, deadlines, and common administrative language.

## Highlights

- Added interactive Dutch vocabulary quiz app.
- Added **238 vocabulary entries** across topics such as:
  - obligation and permission
  - time and deadlines
  - conditions
  - connectors
  - formal verbs
  - documents and procedures
  - work, health, shopping, neighbourhood, and language vocabulary
- Added multiple practice modes:
  - **Invullen** — choose the missing word
  - **Typen** — type the missing word
  - **Puzzel** — unscramble words or phrases
  - **Koppelen** — match Dutch words to translations
  - **Zinsbouw** — rebuild Dutch sentences from shuffled chips
  - **Alles** — combined vertical practice mode
- Added XP, streaks, round summaries, missed-word review, emoji feedback, toast messages, and confetti.
- Added word explanation popup with translation, word-part breakdown, and study tips.
- Added Portuguese support in the UI/support-language layer, with English fallback.
- Added Docker/Nginx runtime for static deployment.
- Added TypeScript local dev server via `npm run dev`.
- Added vocabulary schema documentation and project context documentation.

## Developer Notes

- Static app served from `src/`.
- Main app file: `src/woordenschat_oefeningen_v1.html`.
- Vocabulary data: `src/vocab.json`.
- Vocabulary schema: `docs/VOCABULARY_SCHEMA.md`.
- Docker image uses `nginx:1.27-alpine`.

## Run Locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:8090
```

## Run with Docker

```bash
docker build -t dutch-language .
docker run --rm -p 8080:80 dutch-language
```

Open:

```text
http://localhost:8080
```
